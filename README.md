## Resúmen de funcionalidad

La aplicacion es un sitio de administración de recibos. Utiliza una API personalizada para las solicitudes, incluida la autenticacion.

**Funcionalidad General:**

- Autenticar usuarios a través de JWT
- CRUD Usuarios
- CRUD Recibos
